Este es un programa Realizado en Java que gestiona una matriz de 5x5
Acepta solo Numero enteros
Esta validado para que el usuario NO ingrese datos como Caracteres ni numeros decimales
El usuario debe ingresar por teclado cada elemento de la matriz que se especifica claramente. 
Luego que haya ingresado el programa imprime la matriz en forma de tabla en 5 columnas y 5 filas.
